This plugin extends WooCommerce by allowing a custom product tab to be created with any content.


This plugin extends [WooCommerce](http://woocommerce.com/) to allow multiple custom tabs to be added to single product pages with arbitrary content. The new custom tab may contain text, HTML (such as embedded videos), or shortcodes, and will appear between the "Additional Information" and "Reviews" tabs.

> Requires WooCommerce 3.1.0 or newer

= Features =

 - Add multiple custom tab to each product in your shop
 - Insert any desired content into custom tabs to provide product specifications, shipping info, or more
 - Custom tabs can accept shortcodes or HTML content &ndash; great for embedding a marketing video or inquiry form!

== Installation ==

1. Upload the entire 'cn-custom-woo-tabs' folder to the '/wp-content/plugins/' directory, **or** upload the zip file via Plugins &gt; Add New
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit a product, then click on 'Cn Custom Tab' within the 'Product Data' panel
